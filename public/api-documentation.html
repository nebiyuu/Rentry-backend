Available Endpoints

GET /

Health check endpoint that returns service status and lists all available endpoints.

Parameters

None

Response Example

{
  "message": "Rentry AI Engine is Online",
  "endpoints": [
    "/recommend/{id}",
    "/predict-price/{value}",
    "/validate-image/"
  ]
}
GET /recommend/{product_uuid}

Get product recommendations based on a given product UUID using AI similarity analysis.

Parameters

Name Type Required Description
product_uuid string Required UUID of the product to get recommendations for

Response Example

{
  "status": "success",
  "recommended_ids": ["uuid1", "uuid2", "uuid3"]
}
GET /predict-price/{market_value}

Predict rental price based on market value using AI pricing model with 5-tier fallback logic.

Parameters

Name Type Required Description
market_value float Required Market value of the product

Response Example

{
  "status": "success",
  "suggested_rent": 25.50
}
GET /validate-image/

Validate image quality using computer vision analysis to check for blur, exposure, and other quality metrics.

Parameters

Name Type Required Description
path string Required File path to the image to validate (query parameter)

Example Usage

GET /validate-image/?path=/path/to/image.jpg

Response Example

{
  "is_valid": true,
  "quality_score": 0.85,
  "issues": []
}
